This is the Java-mountains home page

Java-mountains is a port of the xmountains fractal landscape generator to Java.

You can see the code run by viewing this page with a java capable browser.

Or start using Java web-start

You can pause the applet by clicking on the image.

You can also look at the code:

  • Mountain.java The terrain generator.
  • Artist.java The image rendering code.
  • mountapp.java A simple applet.
